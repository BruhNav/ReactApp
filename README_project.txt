How to view the web page-

1. Navigate to the folder "assigment-using-React.js" using command shell
2. Run command "npm start"

About the modules used-

The whole web page is divided into components:

Header.js: contains the header component
ServiceInformation.js: contains the service blocks
blockInfo.json: contains the data of the services like service name, time required and information of the service
App.js: imports all the above listed files and reads the data in json and passes it to ServiceInformation using react props
Index.js: main file that imports app.js component and renders to main webpage
style.css: applies styles to the page
